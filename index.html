🚀 Como Instalar e Usar

Passo 1: Baixar o Projeto

Clone ou baixe todos os arquivos do projeto:

git clone https://github.com/seu-usuario/tiktok-auto-poster.git
cd tiktok-auto-poster
📥 Baixar ZIP

Passo 2: Instalar Python

Certifique-se de ter Python 3.7+ instalado:

python --version

Se não tiver, baixe em: python.org

Passo 3: Instalar Dependências

pip install -r requirements.txt

Ou instale manualmente:

pip install moviepy Pillow gTTS openai flask flask-socketio apscheduler python-dotenv requests numpy opencv-python

Passo 4: Configurar (Opcional)

Crie um arquivo .env para usar IA avançada:

OPENAI_API_KEY=sua_chave_api_aqui

Passo 5: Executar

Versão Terminal (Recomendada):

python tiktok_bot.py

Versão Web:

python app.py
# Acesse: http://localhost:5000

📋 Comandos Rápidos

# Windows (PowerShell)
mkdir tiktok-auto-poster; cd tiktok-auto-poster
pip install moviepy Pillow apscheduler
python -c "open('tiktok_bot.py', 'w').write(open('https://raw.githubusercontent.com/seu-usuario/tiktok-auto-poster/main/tiktok_bot.py').read())"
python tiktok_bot.py

# Mac/Linux
mkdir tiktok-auto-poster && cd tiktok-auto-poster
pip3 install moviepy Pillow apscheduler
curl -O https://raw.githubusercontent.com/seu-usuario/tiktok-auto-poster/main/tiktok_bot.py
python3 tiktok_bot.py

⚠️ Avisos Importantes

  • 🚫 O TikTok não permite uploads automatizados via API pública
  • ⚠️ Automação excessiva pode resultar em banimento da conta
  • ✅ Use com responsabilidade: gere conteúdo, mas publique manualmente
